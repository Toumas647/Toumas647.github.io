## Configuration

Jekyll uses `_config.yml` for settings:

```yaml
# Site settings
title: My GitHub Pages Website
description: A static website
theme: jekyll-theme-slate

# Build settings
markdown: kramdown
```

Simple YAML configuration controls your entire site!
